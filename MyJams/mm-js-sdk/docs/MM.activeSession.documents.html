Namespace: documents

MM.activeSession. documents

MM.activeSession.documents represents the Documents collection related to a session in the MindMeld API. The searchable corpus of Document objects that are contextually related to the Session.

Methods

get(params, onSuccess, onFail)

Get and search across all documents indexed for your application. In addition to providing faceted search and filtering across your collection of documents, this endpoint also provides the capability to deliver relevant document results based on the contextual history of your session. A request with a user token can retrieve documents only if the associated user is permitted to access the session object itself. A request with an admin token can retrieve documents for any session associated with your application.

Name Type Description
params QueryParameters optional

A QueryParameters object allowing you to filter the documents returned. See documentation here for more details For this function, the following additional parameters are also available:

Name Type Description
domainid string optional

optional ID of domain to search documents
entityids Array.<string> | string optional

An array of entityid values or a single entityid value. If specified, only documents related to the specified entities will be returned in the response.
numentities number optional

The number of most recent entities to include in the request. If specified, only documents related to the specified number of most recent entities will be returned in the response.
textentryids Array.<string> | string optional

An array of textentryid values or a single textentryid value. If specified, only documents related to the specified text entries will be returned in the response
query string optional

A search query string to retrieve specific objects that match the query. See the documentation on search query syntax for more information.
document-ranking-factors string optional

A JSON string containing custom factors that will be used to rank the documents returned by this request. Read the section on custom ranking factors to learn more about how you can adjust the search ranking factors to customize the document results for your application.
history-since number | string optional

A Unix timestamp or strtotime date value that specifies the beginning of the contextual history time window that will be used to influence the document results. Any contextual data uploaded prior to the start of this window will be ignored in the calculation to determine contextually relevant document results. If not specified, the value defaults to the latest contextual history of the session.
history-until number | string optional

A Unix timestamp or strtotime date value that specifies the end of the contextual history time window that will be used to influence the document results. Any contextual data uploaded after the end of this window will be ignored in the calculation to determine contextually relevant document results. If not specified, the value defaults to the latest contextual history of the session
start number optional

The index of the first object in the returned list of objects. This can be used for paging through large collections of objects.
limit number optional

The maximum number of individual objects to be returned in the response. If not specified, the default is 10. The maximum allowed value is 50.
since number | string optional

A Unix timestamp or strtotime date value that specifies the start of a range of time-based data. Only documents with publication date after this timestamp will be returned in the response.
until number | string optional

A Unix timestamp or strtotime date value that specifies the end of a range of time-based data. Only documents with publication date before this timestamp will be returned in the response.
onSuccess APISuccessCallback optional

callback for when getting the session document list was successful
onFail APIErrorCallback optional

callback for when getting the session document list failed
Example
         function getDocuments () {
            var queryParams = {
                numentities: 4
            };
            // add custom ranking factors to this query
            queryParams['document-ranking-factors'] = {
                recency: 0.5,
                popularity: 0.7,
                relevance: 0,
                proximity: 0.2,
                customrank1: 0.3
            };
            queryParams['history-since'] = 'yesterday';
            MM.activeSession.documents.get(queryParams, onGetDocuments);
         }
         function onGetDocuments (response) {
            var documents = response.data;
            console.log(documents);
         }

json(){Array.<Object>}

Helper function returns the JSON data for the session documents collection

Example
         function getDocuments () {
            MM.activeSession.documents.get(null, onGetDocuments);
         }
         function onGetDocuments () {
            var documents = MM.activeSession.documents.json();
            // MM.activeSession.documents.json() returns a JSON object
            // containing data received from MM.activeSession.documents.get()
         }

onUpdate(updateHandler, onSuccess, onError, getQueryParams)

Sets the activeSession's documents' onUpdate handler. Pass null as the updateHandler parameter to deregister a previously set updateHandler.

Name Type Description
updateHandler APISuccessCallback optional

callback for when the activeSession's document list updates.
onSuccess function optional

callback for when subscription to onUpdate event succeeds
onError function optional

callback for when subscription to onUpdate event fails
getQueryParams QueryParamGetter optional

custom function used to determine QueryParameters used to in get() request when collection updates
Example
         function documentsOnUpdateExample () {
            // set the onUpdate handler for the documents list
            MM.activeSession.documents.onUpdate(
                onDocumentsUpdate,
                onSubscribedToDocumentsUpdates,
                onSubscribeToDocumentsUpdatesError,
                getDocumentsListParams
            );
         }
         function onSubscribedToDocumentsUpdates () {
            // successfully subscribed to updates to the session's document list
            // now, post a text entry
            createTextEntry();
         }
         function onSubscribeToDocumentsUpdatesError () {
            console.log('error subscribing to document list updates');
         }
         function getDocumentsListParams () {
            // When the document list updates, only fetch 5 objects
            return {
                limit: 5
            };
         }
         function onDocumentsUpdate () {
            // there was an update to the documents list
            var documents = MM.activeSession.documents.json();
            // documents contains the latest list of documents
         }

         function createTextEntry () {
            var textEntryData = {
                text: 'What was the episode where Elaine is banned from the soup shop?',
                type: 'text',
                weight: 1.0
            };
            MM.activeSession.textentries.post(textEntryData);
         }
