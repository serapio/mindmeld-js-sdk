Namespace: voiceNavigator

MM. voiceNavigator

The voice navigator is a widget that allows developers to add voice-driven search to their web applications. By adding a small snippet of JavaScript to your page, you can add our voice navigator to your page allowing your users to search and discover your content in natural, spoken language. The voice navigator widget takes care of capturing speech input from your users, displaying a real-time transcript of what is being recorded, and displaying a collection of results in the browser.

The voice navigator will display when elements with the 'mm-voice-nav-init' class are clicked and when elements with the 'mm-voice-nav-text-init' receive an enter keypress.

See:
  • VoiceNavigatorConfig for full documentation of configuration options.
  • MindMeld Voice Navigator to get started with Voice Navigator.
  • MindMeld Demos to see the Voice Navigator in action.

Examples

Loading the voice navigator 

 <script type="text/js">
 var MM = window.MM || {};
     ( function () {
         MM.loader = {
             rootURL: 'https://www.expectlabs.com/public/sdks/js/'
         ,   widgets: ['voice']
         };
         MM.widgets = {
             config: {
                 appID: 'YOUR APPID'
             ,   voice: voiceNavigatorConfig  // this object contains your configuration options
             }
         };
         var script = document.createElement('script');
         script.type = 'text/javascript'; script.async = true;
         script.src = MM.loader.rootURL + 'embed.js';
         var t = document.getElementsByTagName('script')[0];
         t.parentNode.insertBefore(script, t);
     }());
 </script>

Opens the voice navigator modal window

Name Type Description
options Object optional
Name Type Default Description
query String optional

if provided, this field will be the initial query, and will immediately show results
forceNewIFrame boolean false optional

if true, any voice navigators that have previously been created will be destroyed, and a new instance will be created.
